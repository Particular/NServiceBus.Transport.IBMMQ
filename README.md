# NServiceBus.Transport.IBBMQ

NServiceBus.Transport.IBBMQ enables the use of [IBM MQ](https://www.ibm.com/docs/en/ibm-mq/9.4.x) as the underlying transport for NServiceBus. This transport uses the [IBMMQDotnetClient](https://www.nuget.org/packages/IBMMQDotnetClient)

It is part of the [Particular Service Platform](https://particular.net/service-platform), which includes [NServiceBus](https://particular.net/nservicebus) and tools to build, monitor, and debug distributed systems.

See the [IBMMQ Transport documentation](https://docs.particular.net/transports/IBMMQ) for more details on how to use it.
